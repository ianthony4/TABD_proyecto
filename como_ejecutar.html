🚀 Guía de Ejecución: Test Harness Local

Esta guía te mostrará paso a paso cómo probar y ejecutar el entorno de evaluación de recuperación de información en tu propia computadora.

1 Instalar Dependencias

Primero, asegúrate de tener Python 3.10+ instalado. Abre tu terminal de comandos, navega a la carpeta del proyecto e instala los requerimientos:

cd ~/TABD/proyecto_investigacion
pip install -r requirements.txt

Nota: Las dependencias incluyen pandas (para los CSV), chromadb y sentence-transformers (para el motor vectorial), y pdfplumber (para leer los PDF).

2 Colocar tu Documento (Opcional)

El código generará datos "dummy" la primera vez, pero para hacer la prueba real, coloca tu archivo PDF médico dentro de la ruta:

data/pdfs/documento_minsa_dummy.pdf

Si tu PDF se llama diferente, asegúrate de actualizar el nombre en el archivo main.py (alrededor de la línea 12).

3 Ejecutar el Experimento

Por defecto, el código principal evalúa el MVP A (RAG Vectorial). Para correrlo, ejecuta el script principal en la terminal:

python main.py

La consola te irá indicando cómo el sistema indexa el PDF, lee las preguntas y genera las respuestas. Al finalizar, busca los resultados en data/results.csv.

4 ¿Cómo probar el MVP B (Tree Reasoning)?

Una vez que las "Piezas de Rompecabezas" de la API de PageIndex estén configuradas (ver README.md), puedes evaluar el MVP B editando el archivo main.py. Busca estas líneas:

# Usa esto para RAG
mvp_a = VectorRAGController(chunk_size=500, overlap=100, top_k=2)
orchestrator = ExperimentOrchestrator(retriever=mvp_a)

Y reemplázalas por:

from src.mvp_b_vectorless.tree_controller import TreeReasoningController

# Usa esto para Tree Reasoning
mvp_b = TreeReasoningController()
orchestrator = ExperimentOrchestrator(retriever=mvp_b)
🧩 Información Importante: El código actual está operando en "Modo Simulación" para el LLM y el MVP B. Las respuestas que verás en el CSV dirán "[LLM Mock]" o "[MVP B Contexto Simulado]" hasta que introduzcas las llaves API reales en llm_client.py y tree_controller.py.
